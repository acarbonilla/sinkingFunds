Tutorial about forms
source: https://www.youtube.com/watch?v=VYs-u0g__1A&t=210s&ab_channel=PrettyPrinted
1. Install tweaks
    > pip install django-widget-tweaks
2. Add the tweaks to settings installed apps
    > 'widget_tweaks',
3. Inside the template
    {% load widget_tweaks %}
4. Use rendered field
    This is on template
        > {% render_field form.name class="input" placeholder="" %}
